# Steigerung - So sánh

Bài tập về so sánh (Komparativ và Superlativ) trong tiếng Đức

## 📚 Nội dung chương trình

### 3 bài tập chính:

1. **Teil 1: So sánh bằng** - Gleichheit (so...wie, genauso...wie)
2. **Teil 2: So sánh hơn (Komparativ)** - schöner als, größer als
3. **Teil 3: So sánh nhất (Superlativ)** - am schönsten, der größte

## 🎯 Mục tiêu học tập

- Học cách so sánh bằng, hơn và nhất trong tiếng Đức
- Phân biệt khi nào dùng "als" và "wie"
- Chia đuôi Superlativ đúng cách
- Nhớ các tính từ bất quy tắc (gut-besser-am besten)

## 📖 Cách học hiệu quả

1. **Học công thức từng loại** - so...wie, -er als, am -sten
2. **Nhớ tính từ bất quy tắc** - gut/besser/am besten, viel/mehr/am meisten
3. **Thực hành với câu thực tế** - Không chỉ học thuộc mà phải áp dụng
4. **Chú ý đuôi Superlativ** - der/die/das + -ste vs am + -sten
