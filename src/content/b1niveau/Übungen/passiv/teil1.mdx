Teil 1: Passiv Präsens mit Modalverben
1. Das Zimmer / sauber machen / müssen
→ Das Zimmer ___ [muss sauber gemacht werden.]
2. Die Hausaufgaben / bis morgen / machen / sollen
→ Die Hausaufgaben ___ [sollen bis morgen gemacht werden.]
3. Der Brief / heute / schicken / müssen
→ Der Brief ___ [muss heute geschickt werden.]
4. Die Wäsche / morgen / waschen / können
→ Die Wäsche ___ [kann morgen gewaschen werden.]
5. Das Auto / sofort / reparieren / müssen
→ Das Auto ___ [muss sofort repariert werden.]
6. Die Prüfung / nächste Woche / schreiben / müssen
→ Die Prüfung ___ [muss nächste Woche geschrieben werden.]
7. Der Vertrag / heute / unterschreiben / müssen
→ Der Vertrag ___ [muss heute unterschrieben werden.]
8. Die Fenster / jeden Tag / putzen / sollen
→ Die Fenster ___ [sollen jeden Tag geputzt werden.]
9. Die Pakete / morgen / liefern / können
→ Die Pakete ___ [können morgen geliefert werden.]
10. Der Computer / heute / installieren / müssen
→ Der Computer ___ [muss heute installiert werden.]
11. Die Tickets / im Voraus / buchen / sollen
→ Die Tickets ___ [sollen im Voraus gebucht werden.]
12. Das Essen / schnell / servieren / müssen
→ Das Essen ___ [muss schnell serviert werden.]
13. Die Gäste / freundlich / begrüßen / sollen
→ Die Gäste ___ [sollen freundlich begrüßt werden.]
14. Die Regeln / von allen / befolgen / müssen
→ Die Regeln ___ [müssen von allen befolgt werden.]
15. Die Möbel / am Wochenende / liefern / können
→ Die Möbel ___ [können am Wochenende geliefert werden.]
16. Die Straße / im Sommer / reparieren / müssen
→ Die Straße ___ [muss im Sommer repariert werden.]
17. Der Text / noch einmal / lesen / sollen
→ Der Text ___ [soll noch einmal gelesen werden.]
18. Die Bücher / rechtzeitig / zurückgeben / müssen
→ Die Bücher ___ [müssen rechtzeitig zurückgegeben werden.]
19. Die Tür / leise / schließen / sollen
→ Die Tür ___ [soll leise geschlossen werden.]
20. Die Kinder / früh / ins Bett bringen / müssen
→ Die Kinder ___ [müssen früh ins Bett gebracht werden.]
