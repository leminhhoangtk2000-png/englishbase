Teil 1: Passiv Präsens mit Modalverben

1. Das Zimmer / sauber machen / müssen
   → Das Zimmer \_\_\_ [muss sauber gemacht werden.]
2. Die Hausaufgaben / bis morgen / machen / sollen
   → Die Hausaufgaben \_\_\_ [sollen bis morgen gemacht werden.]
3. Der Brief / heute / schicken / müssen
   → Der Brief \_\_\_ [muss heute geschickt werden.]
4. Die Wäsche / morgen / waschen / können
   → Die Wäsche \_\_\_ [kann morgen gewaschen werden.]
5. Das Auto / sofort / reparieren / müssen
   → Das Auto \_\_\_ [muss sofort repariert werden.]
6. Die Prüfung / nächste Woche / schreiben / müssen
   → Die Prüfung \_\_\_ [muss nächste Woche geschrieben werden.]
7. Der Vertrag / heute / unterschreiben / müssen
   → Der Vertrag \_\_\_ [muss heute unterschrieben werden.]
8. Die Fenster / jeden Tag / putzen / sollen
   → Die Fenster \_\_\_ [sollen jeden Tag geputzt werden.]
9. Die Pakete / morgen / liefern / können
   → Die Pakete \_\_\_ [können morgen geliefert werden.]
10. Der Computer / heute / installieren / müssen
    → Der Computer \_\_\_ [muss heute installiert werden.]
11. Die Tickets / im Voraus / buchen / sollen
    → Die Tickets \_\_\_ [sollen im Voraus gebucht werden.]
12. Das Essen / schnell / servieren / müssen
    → Das Essen \_\_\_ [muss schnell serviert werden.]
13. Die Gäste / freundlich / begrüßen / sollen
    → Die Gäste \_\_\_ [sollen freundlich begrüßt werden.]
14. Die Regeln / von allen / befolgen / müssen
    → Die Regeln \_\_\_ [müssen von allen befolgt werden.]
15. Die Möbel / am Wochenende / liefern / können
    → Die Möbel \_\_\_ [können am Wochenende geliefert werden.]
16. Die Straße / im Sommer / reparieren / müssen
    → Die Straße \_\_\_ [muss im Sommer repariert werden.]
17. Der Text / noch einmal / lesen / sollen
    → Der Text \_\_\_ [soll noch einmal gelesen werden.]
18. Die Bücher / rechtzeitig / zurückgeben / müssen
    → Die Bücher \_\_\_ [müssen rechtzeitig zurückgegeben werden.]
19. Die Tür / leise / schließen / sollen
    → Die Tür \_\_\_ [soll leise geschlossen werden.]
20. Die Kinder / früh / ins Bett bringen / müssen
    → Die Kinder \_\_\_ [müssen früh ins Bett gebracht werden.]
