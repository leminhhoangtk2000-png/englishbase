Teil 2: Passiv Präteritum

1. Der Unfall / gestern / melden
   → Der Unfall \_\_\_ [wurde gestern gemeldet.]
2. Die Straße / letzte Nacht / sperren
   → Die Straße \_\_\_ [wurde letzte Nacht gesperrt.]
3. Die Einladungen / vor zwei Wochen / verschicken
   → Die Einladungen \_\_\_ [wurden vor zwei Wochen verschickt.]
4. Der Kuchen / am Sonntag / backen
   → Der Kuchen \_\_\_ [wurde am Sonntag gebacken.]
5. Die Fenster / während des Sturms / beschädigen
   → Die Fenster \_\_\_ [wurden während des Sturms beschädigt.]
6. Das Paket / gestern Nachmittag / zustellen
   → Das Paket \_\_\_ [wurde gestern Nachmittag zugestellt.]
7. Der Film / letztes Jahr / drehen
   → Der Film \_\_\_ [wurde letztes Jahr gedreht.]
8. Die Gäste / freundlich / empfangen
   → Die Gäste \_\_\_ [wurden freundlich empfangen.]
9. Die Brücke / im Winter / renovieren
   → Die Brücke \_\_\_ [wurde im Winter renoviert.]
10. Die Bücher / an die Schüler / verteilen
    → Die Bücher \_\_\_ [wurden an die Schüler verteilt.]
11. Der Vertrag / letzte Woche / verlängern
    → Der Vertrag \_\_\_ [wurde letzte Woche verlängert.]
12. Die Lampen / in allen Zimmern / austauschen
    → Die Lampen \_\_\_ [wurden in allen Zimmern ausgetauscht.]
13. Der Text / gestern Abend / übersetzen
    → Der Text \_\_\_ [wurde gestern Abend übersetzt.]
14. Die Fahrräder / am Bahnhof / stehlen
    → Die Fahrräder \_\_\_ [wurden am Bahnhof gestohlen.]
15. Der Schlüssel / im Büro / finden
    → Der Schlüssel \_\_\_ [wurde im Büro gefunden.]
16. Die Kinder / am Morgen / wecken
    → Die Kinder \_\_\_ [wurden am Morgen geweckt.]
17. Die Präsentation / von der Chefin / halten
    → Die Präsentation \_\_\_ [wurde von der Chefin gehalten.]
18. Der Patient / sofort / operieren
    → Der Patient \_\_\_ [wurde sofort operiert.]
19. Die Ergebnisse / allen Teilnehmern / zeigen
    → Die Ergebnisse \_\_\_ [wurden allen Teilnehmern gezeigt.]
20. Die Blumen / gestern / gießen
    → Die Blumen \_\_\_ [wurden gestern gegossen.]
