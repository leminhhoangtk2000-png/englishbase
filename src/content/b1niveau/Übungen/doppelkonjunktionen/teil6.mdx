# B1 Doppelkonjunktionen - Teil 6

<ExerciseTable>
  <ExerciseHeader>
    <ExerciseInstruction>
      Ứng dụng thực tế - Viết câu hoàn chỉnh
    </ExerciseInstruction>
  </ExerciseHeader>

  <ExerciseBody>
    <ExerciseItem>
      <ExerciseSentence>
        Maria / können / sowohl Klavier spielen / als auch singen
      </ExerciseSentence>
      <ExerciseAnswer>Maria kann sowohl Klavier spielen als auch singen.</ExerciseAnswer>
      <ExerciseExplanation>
        sowohl...als auch = cả hai kỹ năng âm nhạc
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Je / mehr / ich / lesen / desto / besser / verstehen / ich
      </ExerciseSentence>
      <ExerciseAnswer>Je mehr ich lese, desto besser verstehe ich.</ExerciseAnswer>
      <ExerciseExplanation>
        je...desto = đọc nhiều hiểu tốt
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Das Wetter / sein / zwar / schön / aber / zu heiß
      </ExerciseSentence>
      <ExerciseAnswer>Das Wetter ist zwar schön, aber zu heiß.</ExerciseAnswer>
      <ExerciseExplanation>
        zwar...aber = đẹp nhưng nóng
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Wir / haben / weder / Zeit / noch / Geld / für Urlaub
      </ExerciseSentence>
      <ExerciseAnswer>Wir haben weder Zeit noch Geld für Urlaub.</ExerciseAnswer>
      <ExerciseExplanation>
        weder...noch = không có cả hai
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Du / müssen / entweder / Hausaufgaben machen / oder / lernen
      </ExerciseSentence>
      <ExerciseAnswer>Du musst entweder Hausaufgaben machen oder lernen.</ExerciseAnswer>
      <ExerciseExplanation>
        entweder...oder = chọn một việc
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Die Stadt / sein / nicht nur / sauber / sondern auch / sicher
      </ExerciseSentence>
      <ExerciseAnswer>Die Stadt ist nicht nur sauber, sondern auch sicher.</ExerciseAnswer>
      <ExerciseExplanation>
        không chỉ...mà còn (hai ưu điểm)
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Je / früher / wir / losfahren / desto / mehr / Zeit / haben / wir
      </ExerciseSentence>
      <ExerciseAnswer>Je früher wir losfahren, desto mehr Zeit haben wir.</ExerciseAnswer>
      <ExerciseExplanation>
        je...desto = đi sớm có nhiều thời gian
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Sowohl / Lehrer / als auch / Schüler / sein / zufrieden
      </ExerciseSentence>
      <ExerciseAnswer>Sowohl Lehrer als auch Schüler sind zufrieden.</ExerciseAnswer>
      <ExerciseExplanation>
        sowohl...als auch = cả hai đều hài lòng
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Er / trinken / weder / Alkohol / noch / Kaffee
      </ExerciseSentence>
      <ExerciseAnswer>Er trinkt weder Alkohol noch Kaffee.</ExerciseAnswer>
      <ExerciseExplanation>
        weder...noch = không uống cả hai
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Das Handy / sein / zwar / modern / aber / teuer
      </ExerciseSentence>
      <ExerciseAnswer>Das Handy ist zwar modern, aber teuer.</ExerciseAnswer>
      <ExerciseExplanation>
        zwar...aber = hiện đại nhưng đắt
      </ExerciseExplanation>
    </ExerciseItem>

  </ExerciseBody>
