
Teil 6: damit, um ... zu

1. gesund / bleiben / du / Obst / damit / viel / du / essen
   → [Du isst viel Obst, damit du gesund bleibst.]
2. in den Park / um / frische Luft / wir / gehen / haben / zu
   → [Wir gehen in den Park, um frische Luft zu haben.]
3. Bescheid / du / rechtzeitig / ich / planen / geben / können / damit / dir
   → [Ich gebe dir Bescheid, damit du rechtzeitig planen kannst.]
4. seine Grammatik / die Schüler / die Lehrerin / damit / erklären / bestehen / den Test
   → [Die Lehrerin erklärt die Grammatik, damit die Schüler den Test bestehen.]
5. ich / lernen / viel / die Prüfung / um / bestehen / zu
   → [Ich lerne viel, um die Prüfung zu bestehen.]
6. lauter / damit / er / alle / sprechen / hören / ihn
   → [Er spricht lauter, damit alle ihn hören.]
7. wiederholen / neue Wörter / ich / um / nicht / sie / vergessen / zu
   → [Ich wiederhole neue Wörter, um sie nicht zu vergessen.]
8. fahren / damit / langsamer / er / keinen Unfall / er / bauen / nicht
   → [Er fährt langsamer, damit er keinen Unfall baut.]
9. den Text / besser / der Lehrer / damit / verstehen / alle Schüler / erklären / ihn
   → [Der Lehrer erklärt den Text, damit alle Schüler ihn besser verstehen.]
10. in Ruhe / um / lernen / zu / ich / gehen / Bibliothek / die
    → [Ich gehe in die Bibliothek, um in Ruhe zu lernen.]
11. wir / Bahnhof / damit / am / früh / sind / fahren / los / wir / pünktlich
    → [Wir fahren früh los, damit wir pünktlich am Bahnhof sind.]
12. sparen / Geld / Peter / ein neues / kaufen / Auto / um / zu / sein altes / er / verkaufen
    → [Er verkauft sein altes Auto, um ein neues zu kaufen.]
13. wir / morgen / um / sein / gehen / fit / früh / zu / schlafen
    → [Wir gehen früh schlafen, um morgen fit zu sein.]
14. alle / der Lehrer / damit / verstehen / deutlich / er / sprechen / ihn
    → [Der Lehrer spricht deutlich, damit alle ihn verstehen.]
15. ich / Geld / um / Europa / Reise / eine / sparen / mache / nach / zu
    → [Ich spare Geld, um eine Reise nach Europa zu machen.]
