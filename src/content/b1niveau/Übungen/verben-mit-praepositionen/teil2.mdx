# B1 Verben mit Präpositionen - Teil 2: Fragewörter

<ExerciseTable>
  <ExerciseHeader>
    <ExerciseInstruction>
      Đặt câu hỏi phù hợp với **wo(r)-** hoặc **da(r)-** cho mỗi câu sau. -
      **wo(r)-** dùng cho vật/sự việc - **wer/wen/wem** dùng cho người
    </ExerciseInstruction>
  </ExerciseHeader>

  <ExerciseBody>
    <ExerciseItem>
      <ExerciseSentence>Er wartet auf den Bus. → ___?</ExerciseSentence>
      <ExerciseAnswer>Worauf wartet er?</ExerciseAnswer>
      <ExerciseExplanation>
        "warten auf" + vật → "worauf". Wo + r + auf (vì "auf" bắt đầu bằng
        nguyên âm).
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Maria interessiert sich für Musik. → ___?
      </ExerciseSentence>
      <ExerciseAnswer>Wofür interessiert sich Maria?</ExerciseAnswer>
      <ExerciseExplanation>
        "sich interessieren für" + vật → "wofür". Wo + für.
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Wir sprechen über das neue Projekt. → ___?
      </ExerciseSentence>
      <ExerciseAnswer>Worüber sprecht ihr?</ExerciseAnswer>
      <ExerciseExplanation>
        "sprechen über" + vật → "worüber". Wo + r + über (vì "über" bắt đầu bằng
        nguyên âm).
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>Anna denkt an ihre Zukunft. → ___?</ExerciseSentence>
      <ExerciseAnswer>Woran denkt Anna?</ExerciseAnswer>
      <ExerciseExplanation>
        "denken an" + vật → "woran". Wo + r + an (vì "an" bắt đầu bằng nguyên
        âm).
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Ich freue mich auf deinen Besuch. → ___?
      </ExerciseSentence>
      <ExerciseAnswer>Worauf freust du dich?</ExerciseAnswer>
      <ExerciseExplanation>
        "sich freuen auf" + vật → "worauf". Wo + r + auf.
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Er freut sich über das Geschenk. → ___?
      </ExerciseSentence>
      <ExerciseAnswer>Worüber freut er sich?</ExerciseAnswer>
      <ExerciseExplanation>
        "sich freuen über" + vật → "worüber". Wo + r + über.
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>Sie nimmt an einem Kurs teil. → ___?</ExerciseSentence>
      <ExerciseAnswer>Woran nimmt sie teil?</ExerciseAnswer>
      <ExerciseExplanation>
        "teilnehmen an" + vật → "woran". Wo + r + an.
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>
        Peter träumt von einem Haus am Meer. → ___?
      </ExerciseSentence>
      <ExerciseAnswer>Wovon träumt Peter?</ExerciseAnswer>
      <ExerciseExplanation>
        "träumen von" + vật → "wovon". Wo + von (không cần "r" vì "von" bắt đầu
        bằng phụ âm).
      </ExerciseExplanation>
    </ExerciseItem>

    <ExerciseItem>
      <ExerciseSentence>Er kümmert sich um die Kinder. → ___?</ExerciseSentence>
      <ExerciseAnswer>Worum kümmert er sich?</ExerciseAnswer>
      <ExerciseExplanation>
        "sich kümmern um" + vật → "worum". Wo + r + um (vì "um" bắt đầu bằng

    </ExerciseItem>

      <ExerciseExplanation>
