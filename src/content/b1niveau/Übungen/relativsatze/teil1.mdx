# B1 Relativsätze - Teil 1: Relativpronomen

## Lý thuyết ngắn gọn

**Relativpronomen** (đại từ quan hệ) nối câu chính với câu phụ:
- **der, die, das** (Nominativ) - chủ ngữ
- **den, die, das** (Akkusativ) - tân ngữ trực tiếp
- **dem, der, dem** (Dativ) - tân ngữ gián tiếp  
- **dessen, deren, dessen** (Genitiv) - sở hữu
