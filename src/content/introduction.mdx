## What is this?

This is a demo documentation site built with Next.js, Tailwind CSS, and MDX. It's designed to be a starting point for your own documentation.

## How to use it

You can add new pages by creating `.md` or `.mdx` files in the `src/content` directory. The navigation is automatically generated based on the file structure.

For example, this file is located at `src/content/introduction.mdx` and its route is `/docs/introduction`.

To learn more about the features, check out the "Getting Started" section in the sidebar.
