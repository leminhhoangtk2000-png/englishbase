# Welcome to DocuNext

This is a demo documentation page built to showcase the capabilities of this template. You can write your content using Markdown for easy formatting or MDX for embedding interactive React components.

## Key Features

- **Next.js App Router**: Built on the latest Next.js features for optimal performance and developer experience.
- **MDX Support**: Seamlessly blend Markdown with React components to create rich, dynamic content.
- **File-based Routing**: The navigation and routing are automatically generated from your file structure in the `content` directory.
- **Full-text Search**: Instant, client-side search to help users find what they need quickly.
- **Styled with ShadCN UI**: A beautiful, accessible, and customizable component library.

## Getting Started

To add a new page, simply create a new `.mdx` file in the `src/content` directory. The navigation will update automatically.
